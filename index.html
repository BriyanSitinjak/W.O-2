Deskripsi Warteg Online

Warteg Online dibuat untuk memudahkan para user atau mahasiswa untuk mencari makanan dan memesan nya melalui aplikasi ini dan membantu untuk para penjual makanan khususnya untuk makanan berat dalam menjual makanannya supaya lebih cepat terjual makanan nya. Dikareanakan sering dilihat para mahasiswa malas untuk makan karena malas jalan atau malas untuk keluar maka aplikasi ini dapat menangani masalah yang ada di kebiasaan buruk para mahasiswa
